Uninstall eMedia

Back

If you want to uninstall eMedia, you can do just as any other program: DO NOT delete the files in the application directory, as registry informations and system files will not be deleted.

  • Open the Start menu. Select Settings, then Control Panel. Double-click on the Add/Remove programs icon.
  • Locate in the list the entry eMedia card designer, then click on the Add/Remove button.
  • Answer Yes to the question "Do you want to remove completely this application and all its components?".

eMedia uninstalls itself and displays at the very end a message indicating that it is uninstalled.
