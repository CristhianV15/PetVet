Installation process

Back

eMedia uses for database access in professional version and standard version in demo mode Microsoft ActiveX Data Objects™ and OLE DB. But, as ADO is a DCOM component, your system must be up-to-date. The eMedia setup program checks for out-of-date components and gives you the ability to update your computer.

The setup process uses three programs from autorun:

Autorun.exe Launched by Windows when you insert the CD. It will help you to install the printer driver and/or eMedia Card Designer.
eMedia\Setup.exe Performs steps 1 to 9 below.
eMedia\Setup\Setup.exe Perform steps 10 to 12 below.

The setup program performs the following tasks:

1.  On a Microsoft Windows 95™ system, the setup program checks if DCOM for Windows 95 version 1.3 is installed.
On a Windows 98™ or Windows 98 SE™ system , the setup program checks if DCOM for Windows 98 version 1.3 is installed.
DCOM version 1.3 is used by MDAC and eMedia.
2.  The setup program checks if Microsoft Data Access Components™ version 2.5 is installed. MDAC 2.5 is used by eMedia.
3.  The setup program checks if Microsoft Windows Installer™ is installed.
4.  A window is displayed if one or more of these components is not installed.
5.  On a Microsoft Windows NT™ 4.0 or Windows 2000™ system, the setup program displays an alert to prevent the user to install the components without administrative rights, as these installations will fail.
6.  The setup program checks if Microsoft Internet Explorer™ version 4.01+ is installed. If not, an alert is displayed as all the features of the eMedia online help system will only be available with a version of IE greater than 4.01.
7.  If DCOM is not installed, or if its version is lower than 1.3, the setup program installs DCOM for Windows 95 (on a Microsoft Windows 95™ system) or DCOM for Windows 98 (on a Microsoft Windows 98™ system or on a Microsoft Windows 98 Second Edition™ system). Then, the setup program registers itself to be restarted with Windows and displays a message asking the user to reboot the computer. When the system will restart, the setup program will be launched automatically. If it doesn't, the user will have to do it by himself.
8.  If MDAC is not installed, or if its version is lower than 2.5, the setup program installs the component. This installation can take a very long time (more than 10 minutes). The user must not abort the installation. If an error occurs, a message will be displayed. If MDAC needs a reboot, the setup program will ask the user at the very end of the installation process.
9.  If Microsoft Windows Installer™ is not installed, the setup program installs this component.
10. The user can choose the default language that will be used in eMedia.
11.  Then, the setup program uses Microsoft Windows Installer™ to install eMedia.
12.  Once eMedia installed, if restarting the system is needed (by the DCOM or MDAC installations), a window displays an alert, asking the user to reboot.

The entire process is automatic. The user has only to accept the default choices by clicking on the "Next" button.
